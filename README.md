### Mode 模式
- `ESC` 返回普通模式
  
### Move 移動
- `h`: 光標左移
- `j`: 光標下移
- `k`: 光標上移
- `l`: 光標右移
- `w`: 光標前移一個單詞
- `b`: 光標後移一個單詞
- `e`: 光標移至當前單詞末尾
- `$`: 光標移至行尾
- `0`: 光標移至行首
- `gg`: 光標移至文件開始位置
- `G`: 光標移至文件末尾位置
- `}`: 跳轉下一段落
- `{`: 跳轉上一段落
- `:<line_number>`： 跳轉到特定行 `:5`
- `ctrl+f`: 前一屏
- `ctrl+b`: 後一屏
- `ctrl+u`: 向上半頁
- `ctrl+d`: 向下半頁

### Edit 編輯操作
- `i`: 插入模式
- `a`: 在當前字符後進入插入模式
- `x`: 刪除光標下方的字符
- `dd`: 刪除整行內容

### Save & Exit
- `:w`: 保存
- `:q`: 退出
- `:q!`: 不保存退出 
- `:wq`： 保存並退出

### Search 搜索
- `/pattern`:向前搜索單詞`pattern`
- `?pattern`: 向後搜索單詞`pattern`
- `n`: 上一個匹配項
- `N`: 下一個匹配項
